Upload Your Model

Upload a STAC-compliant JSON file with your model's metadata. We’ll extract all necessary fields automatically.
If any data is missing, we’ll guide you to fill in the required information.

Sample JSON Structure

Ensure your JSON file follows this structure:

{
    "type": "Feature",
    "properties": {
        "mlm:name": "Example Model Name",
        "description": "Brief description of your model's purpose and usage.",
        "mlm:tasks": ["classification", "object detection"],
        "mlm:input": [{
            "input": {
                "data_type": "float32",
                "shape": [-1, 3, 224, 224]
            }
        }]
    },
    "assets": {
        "model": {
            "href": "path/to/model.pt",
            "type": "application/octet-stream",
            "description": "PyTorch model file."
        },
        "source_code": {
            "href": "https://github.com/user/repo",
            "type": "text/x-python",
            "description": "Link to the model's source code."
        }
    },
    "links": [
        {
            "rel": "derived_from",
            "href": "https://data.source.url",
            "type": "application/json"
        }
    ],
    "geometry": {
        "type": "Polygon",
        "coordinates": [
            [[-10.0, 50.0], [10.0, 50.0], [10.0, -50.0], [-10.0, -50.0], [-10.0, 50.0]]
        ]
    }
}
Ensure the file adheres to the STAC metadata format. Invalid files will require additional manual data entry.
